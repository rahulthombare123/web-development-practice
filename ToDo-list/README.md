# ✅ Todo List App

A clean, modern, and fully interactive todo list application built with HTML, CSS, and JavaScript. It helps you manage daily tasks with ease — add, edit, delete, complete, and filter tasks. All your data is saved locally in the browser.

## 🔗 Live Demo
👉 https://rahulthombare123.github.io/web-development-practice/ToDo-list

## 🚀 Features

- **Add new tasks** – type and hit enter or click the + button.
- **Edit tasks** – click the pencil icon to modify any task.
- **Delete tasks** – remove individual tasks with the trash icon.
- **Mark as completed** – click the checkbox to toggle task status.
- **Filter tasks** – view All, Active (incomplete), or Completed tasks.
- **Clear completed** – remove all done tasks with one click.
- **Persistent storage** – tasks are saved in your browser’s local storage, so they remain after refresh.
- **Responsive design** – works on desktop, tablet, and mobile devices.
- **Live task counter** – shows the number of remaining active tasks.

## 🛠️ Technologies Used

- **HTML5** – structure and semantic markup.
- **CSS3** – custom styling, Flexbox, gradients, and modern UI.
- **JavaScript (ES6)** – dynamic interactions, DOM manipulation, and local storage.
- **Font Awesome 6** – icon library for intuitive controls.
- **Google Fonts (Inter)** – clean, readable typography.


## 🚀 How to Run

1. **Clone or download** the repository.
2. Open the `index.html` file in any modern web browser.
3. Start adding tasks!

No build steps, no dependencies — just open and use.

## 📸 Usage Preview

1. **Add a task**  
   Type your task in the input field and click the **+** button or press **Enter**.

2. **Complete a task**  
   Click the checkbox next to any task. Completed tasks are styled with a strikethrough.

3. **Edit or delete**  
   Hover over a task to reveal the edit (✏️) and delete (🗑️) icons.

4. **Filter tasks**  
   Use the **All**, **Active**, or **Done** buttons to view specific task sets.

5. **Clear completed**  
   Click **Clear completed** to remove all finished tasks at once.

## 🧠 Local Storage

All tasks are saved in `localStorage` under the key `todo_list_app`..  
Even after closing the browser, your tasks will be there when you return.

## 🎨 Customization

- Change the primary color scheme by editing the CSS gradients and button colors in the `<style>` section.
- Modify the placeholder text or labels as needed.
- Adjust the max-height of the task list to fit your design.
