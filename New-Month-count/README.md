# 🌟 New Month Countdown Timer

A simple and elegant **Countdown Timer Web App** that displays the remaining time (Days, Hours, Minutes, Seconds) until the start of the next month.

🚀 Built using **HTML, CSS, and JavaScript**

---

### 🔗 Live Demo

👉 [https://rahulthombare123.github.io/web-development-practice/New-Month-count/](https://rahulthombare123.github.io/web-development-practice/New-Month-count/)

---

## ✨ Features

* ⏳ Real-time countdown timer
* 📅 Displays:

  * Days
  * Hours
  * Minutes
  * Seconds
* ⚡ Updates every second automatically
* 🎨 Clean and modern UI design
* 📱 Fully responsive
* 🔄 Can be modified for any target date

## ⚙️ How It Works

* The app calculates the difference between:

  * Current Date & Time
  * Target Date (Next Month)
* Converts the difference into:

  * Days
  * Hours
  * Minutes
  * Seconds
* Updates the UI every second using:

```javascript
setInterval(timer, 1000);
```
