# 🌀 CSS Loader Animation

A clean, modern, and fully responsive loading animation built with pure HTML and CSS. Perfect for use as a splash screen, placeholder, or loading indicator in any web project.

## ✨ Features

- **Pure CSS** – no JavaScript, no external dependencies.
- **Smooth animations** – spinning circle + pulsing text.
- **Modern design** – translucent glassmorphism container with backdrop blur.
- **Responsive** – scales beautifully on all screen sizes.
- **Easy to customize** – change colors, sizes, and speeds in seconds.

## 🛠️ Technologies Used

- HTML5
- CSS3 (animations, flexbox, backdrop-filter)
- Google Fonts (Segoe UI fallback)

## 🚀 How to Use

1. Copy the entire HTML code into a new file (e.g., `loader.html`).
2. Open the file in any modern web browser.
3. To integrate into your own website, copy the `<div class="loader-container">` block and the associated CSS into your project.

## 🎨 Customization

- **Colors**: Modify the `background` of `body`, the `border` colors of `.spinner`, and the `color` of `.loading-text`.
- **Speed**: Adjust the `animation` durations (e.g., `1s` for spin, `1.5s` for pulse).
- **Size**: Change `width` and `height` of `.spinner` and `font-size` of `.loading-text`.
- **Container**: Remove or tweak the `backdrop-filter` and `background` of `.loader-container` for a different look.
