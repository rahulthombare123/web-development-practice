# Animated CSS Buttons Collection

A set of ten CSS buttons that change color, size, or shape on hover.  
Perfect for spicing up any web project — no JavaScript required.

## ✨ Features

- 10 unique hover effects (color swap, grow, shrink, border pop, glow, rotate, invert, stretch, vibrant, morph)
- Smooth CSS transitions
- Fully responsive
- Easy to integrate or customize


## 🛠️ Usage

1. Copy the `<button>` HTML you like.
2. Add the corresponding CSS class (e.g., `btn-grow`) to your stylesheet.
3. Adjust colors, sizes, or transition timings to match your brand.

## 🎨 Button Styles

| Class name        | Hover effect                              |
|-------------------|-------------------------------------------|
| `btn-color-swap`  | background & text color swap + lift       |
| `btn-grow`        | scale up (1.15) + darker background       |
| `btn-shrink`      | scale down (0.9) + color change           |
| `btn-border-color`| border & text color change, subtle scale  |
| `btn-glow`        | deep shadow + translateY                   |
| `btn-pop`         | scale + rotate + brighten                  |
| `btn-invert`      | invert colors + slight grow                |
| `btn-stretch`     | horizontal padding expansion               |
| `btn-vibrant`     | scale + background change + outline glow   |
| `btn-morph`       | border radius morph + color swap           |
